





### kubecm



```
Kubecm 是一个用于管理和切换 kubeconfig 的工具

Usage:
  kubecm [flags]
  kubecm [command]

Available Commands:
  add         新增配置
  del         删除配置
  desc        打印配置
  list        列出配置
  rename      配置更名
  switch      切换配置
  version     版本信息
```
